An engine in the style of Eye of the Beholder / Dungeon Master: a four-character party explores a tile-based dungeon rendered in first-person pixel art, moving one square at a time with 90° turns, fighting monsters in pseudo-real-time, and reading about all of it in a scrolling message log.

This document is the design + implementation plan. Sections 1–7 describe the design; Section 8 is the milestone-by-milestone build order.
